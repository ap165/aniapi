## Search Anime BY Name

`
url:-
    /api/search

params:-
    q= [anime-name]
    page= [page-no]
`
```bash
http://localhost:5000/api/search?q=naruto&page=1
```

## AJAX Search
`
url:- 
    /ajax/search

params:-
    q= [anime-name]
`
```bash
http://localhost:5000/api/ajax/search?q=naruto
```
